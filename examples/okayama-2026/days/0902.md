# 09/02（Day 6）尾道市區深度日

- **住宿：** 尾道

## 時間表

| 時間 | 動作 |
|---|---|
| 07:00–07:30 | 早餐：尾道站／本通商店街周邊超商或麵包店先行解決（公寓式住宿不含早餐，請自理）。 |
| 07:30–08:45 | 晨間漫步[尾道水道](https://www.google.com/maps/search/?api=1&query=尾道水道)海岸線與[尾道本通商店街](https://www.google.com/maps/search/?api=1&query=尾道本通商店街)入口段：店家多數未開、街景寧靜，碼頭邊望著渡輪往返於向島之間，與午後喧鬧形成對比，是感受尾道港町晨間風情的最佳時刻。 |
| 08:45–09:00 | 步行至[千光寺山纜車](https://www.google.com/maps/search/?api=1&query=千光寺山ロープウェイ+尾道)搭乘處（約 15 分），提早抵達準備搭乘首班車。 |
| 09:00–09:30 | 搭乘首班纜車上山（**09:00 首發，每 15 分一班**；片道 ¥500 / 往復 ¥700）→ [千光寺公園展望台](https://www.google.com/maps/search/?api=1&query=千光寺公園展望台+尾道)眺望尾道水道與向島：搭頭班車可避開稍後湧入的團客人潮，晨光下視野最為通透。 |
| 09:30–10:30 | [千光寺](https://www.google.com/maps/search/?api=1&query=千光寺+尾道)參拜，沿[文學小徑](https://www.google.com/maps/search/?api=1&query=文学のこみち+尾道)石板路往下走。 |
| 10:30–11:30 | [貓之細道](https://www.google.com/maps/search/?api=1&query=猫の細道+尾道)漫步，尋找沿牆的招財貓石像，探索山腰小巷。 |
| 11:30–13:00 | 下山至[尾道本通商店街](https://www.google.com/maps/search/?api=1&query=尾道本通商店街)，午餐（尾道拉麵或老屋咖啡）。 |
| 13:00–17:00 | 商店街深度逛：老屋咖啡、創作雜貨、二手書店；可安排林芙美子文學路線。 |
| 17:00–17:45 | 沿[尾道水道](https://www.google.com/maps/search/?api=1&query=尾道水道)海岸步道散步，眺望對岸向島——夕照水岸與清晨所見的寧靜畫面相互呼應，為一日做美好收尾。 |
| 18:00–19:00 | 晚餐（推薦海鮮或居酒屋，換換口味）。 |

## 行程

- [千光寺山纜車](https://www.google.com/maps/search/?api=1&query=千光寺山ロープウェイ+尾道)｜從山腳到山頂僅 3 分鐘，上山省力又有俯瞰港口的好視角；片道 ¥500、往復 ¥700。**全年營運 09:00–17:15，每 15 分一班，09:00 為首班車**（這也是今天行程安排在 09:00 才上山的原因，09:00 前的空檔改以晨間水道散步運用）
- [千光寺公園展望台](https://www.google.com/maps/search/?api=1&query=千光寺公園展望台+尾道)｜尾道水道、因島與向島盡收眼底，也是確認明天島波海道路線的好地點
- [千光寺](https://www.google.com/maps/search/?api=1&query=千光寺+尾道)｜紅色本堂嵌入岩壁，從展望台沿石板「文學小徑」步行下山至商店街
- [貓之細道](https://www.google.com/maps/search/?api=1&query=猫の細道+尾道)｜山腰窄巷，沿途有藝術家手繪貓咪圖案石頭，是尾道著名打卡路線
- [尾道本通商店街](https://www.google.com/maps/search/?api=1&query=尾道本通商店街)｜全長約 1.2 公里的有頂棚老商店街，新舊交雜，藏有老屋咖啡館與創作雜貨

**備案：** 若天氣不佳，可安排室內行程：[尾道市立美術館](https://www.google.com/maps/search/?api=1&query=尾道市立美術館)（千光寺公園內），或在商店街老屋咖啡館消磨半日。

> ⚠️ 千光寺山纜車有固定維修停駛期，請確認當日是否正常運行。
