# 09/01（Day 5）尾道出發 → 倉敷日歸（含大原美術館）

- **住宿：** 尾道
- **本日為週二，大原美術館正常開館**（09:00–17:00，¥2,000）

## 交通

| 區間 | 方式 | 時間 | 費用 |
|------|------|------|------|
| 尾道站 → 倉敷站 | JR 山陽本線（往岡山方向） | 約 65–70 分鐘 | 約 ¥1,170 |
| 倉敷站 → 尾道站 | JR 山陽本線（往三原方向） | 約 65–70 分鐘 | 約 ¥1,170 |

> ※ 來回交通費合計約 ¥2,340。建議使用 ICOCA 感應，無需每次購票排隊。

## 時間表

| 時間 | 動作 |
|---|---|

| 06:10–06:20 左右 | 搭 JR 山陽本線往岡山方向（車程約 65–70 分鐘）。※ 請出發前確認當日班次 |
| 07:15–07:25 左右 | 抵達[倉敷站](https://www.google.com/maps/search/?api=1&query=JR倉敷駅)，步行至美觀地區（約 15 分）。 |
| 07:30–08:45 | 晨間漫步[倉敷美觀地區](https://www.google.com/maps/search/?api=1&query=倉敷美観地区)：晨光下的白壁倉庫與運河街景，店家尚未開門、遊客寥寥，正是拍照與感受古街氛圍的最佳時刻——比昨日（08/31）匆忙一瞥更從容盡興。 |
| 08:45–09:00 | 移動至大原美術館入口，候場準備入館。 |
| 09:00–11:30 | [大原美術館](https://www.google.com/maps/search/?api=1&query=大原美術館+倉敷)（09:00 開館，¥2,000；建議預留 2–2.5 小時）。 |
| 11:30–13:00 | [倉敷美觀地區](https://www.google.com/maps/search/?api=1&query=倉敷美観地区)午餐＋散步（運河旁拍照、倉庫街）。 |
| 13:00–15:30 | 倉敷深度探訪：[倉敷帆布](https://www.google.com/maps/search/?api=1&query=倉敷帆布)・[林源十郎商店](https://www.google.com/maps/search/?api=1&query=林源十郎商店+倉敷)・[倉敷常春藤廣場](https://www.google.com/maps/search/?api=1&query=倉敷アイビースクエア)。 |
| 15:30 | 前往倉敷站準備回程。 |
| 15:45–16:00 左右 | [倉敷站](https://www.google.com/maps/search/?api=1&query=JR倉敷駅)搭 JR 山陽本線往尾道（約 65–70 分鐘）。※ 請確認班次 |
| 17:00–17:10 左右 | 抵達[尾道站](https://www.google.com/maps/search/?api=1&query=JR尾道駅)，返回飯店。 |
| 18:30–19:30 | 晚餐（尾道本通商店街附近）。 |

## 行程

- [大原美術館](https://www.google.com/maps/search/?api=1&query=大原美術館+倉敷)｜日本西洋美術收藏先驅，莫內《睡蓮》、畢卡索、羅丹等原作；09:00–17:00，¥2,000。本旅程唯一有機會進館的時機，務必優先安排
- [倉敷帆布](https://www.google.com/maps/search/?api=1&query=倉敷帆布)｜倉敷特產厚帆布製品（托特包、錢包等），工藝紮實，適合買做伴手禮
- [林源十郎商店](https://www.google.com/maps/search/?api=1&query=林源十郎商店+倉敷)｜老藥局改建的生活雜貨複合店，選品精緻，適合慢慢挑
- [倉敷美觀地區](https://www.google.com/maps/search/?api=1&query=倉敷美観地区)｜昨日未及深逛的角落今日從容補完

**替代方案（下午）：** 若對丹寧文化有興趣，可改去[兒島牛仔褲街](https://www.google.com/maps/search/?api=1&query=児島ジーンズストリート)（倉敷站搭 JR 瀬戸大橋線至兒島站約 15 分，¥330）；返程時再回倉敷轉尾道。

> ⚠️ 大原美術館出發前請至[官網](https://ohara.or.jp/)確認開館時間。具名餐廳請出發前確認最新營業時間與公休日。
