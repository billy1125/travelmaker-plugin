# 岡山桃太郎機場 ↔ 岡山站 利木津巴士指南

## 路線概覽

| 項目 | 內容 |
|------|------|
| 路線 | 岡山桃太郎機場（2號站牌）↔ 岡山站西口（21號站牌） |
| 所需時間 | 約 **30 分鐘**（視交通狀況，山路段可能延誤） |
| 票價（岡山站） | 大人 **1,000 円**・兒童 **500 円** |
| 票價（倉敷站） | 大人 **1,400 円**・兒童 **700 円** |
| 預約 | **不需預約**，直接前往站牌搭車 |
| 營運公司 | 岡電巴士・中鉄巴士（共同運行） |
| 班次頻率 | 約每 30～60 分鐘一班（視時段，詳見時刻表） |

> ⚠️ 票價資訊以出發前官方公告為準。部分中途站（スポーツセンター前・岡山大学筋等）票價較低（950円）。

## 站牌位置

- **機場出發**：到達大廳出口外右手邊，[2號站牌](https://www.google.com/maps/search/?api=1&query=岡山桃太郎空港バス乗り場)，出門即可看到售票機與工作人員
- **岡山站出發**：岡山站西口，[21號站牌](https://www.google.com/maps/search/?api=1&query=岡山駅西口バスのりば)（候車室內設有售票機與廁所，有座位）

## 付款方式

| 方式 | 說明 |
|------|------|
| 全國交通系 IC 卡 | ICOCA、Suica、HARECA 等，上下車時感應 |
| 售票機（機場 / 岡山站西口候車室） | 現金、信用卡（**僅感應式**）、電子錢包、QR 碼 |
| 車內現金 | 中途站上車者可用現金，整理券制 |

> 建議在售票機提前購票，下車時將票交給司機。

## 搭乘流程（機場出發）

1. 出境後往右走，機場大門外即為巴士乘車區
2. 在售票機選擇目的地（岡山站 / 倉敷站）購票
3. 將大行李交給現場工作人員（**有專人幫忙放置行李艙**）
4. 對號上車等待發車

> 工作人員親切，流程非常直覺。若不確定，跟著人群走即可。

## 國際線加班車（重要）

機場針對 **台北・高雄・首爾・上海** 等國際航班，會在落地後加開臨時接駁巴士，**不列入一般時刻表**。

- 本行程去程 **IT214**（台北 → 岡山，15:05 落地）即屬適用航班
- 落地後請盡快通關（入境審查 + 提領行李 + 海關），直接前往 2 號站牌
- 若人數太多、滿車，會另備備用車載送，**不用擔心搭不上**

## 本次行程使用規劃
