## 規則

- 每個條目必須有 URL 連結、整理日、適用範圍、3–5 條摘要重點
- 摘要重點寫「對旅程有用的事實」，不寫廣告語或空泛描述
- 同一 URL 只記錄一次，若資料有更新則在原條目下補充（加 `**更新日：**` 標記）
- 分類依據：
  - **交通**：機場、巴士、鐵路、渡船、租車等移動方式
  - **美食**：餐廳、咖啡館、在地特色食物
  - **購物**：伴手禮、商店街、百貨
  - **景點**：觀光地點、自然景觀、博物館、文化設施
- 若一個網站同時涵蓋多類（例如綜合攻略），選最主要的分類，並在適用範圍欄位說明
- 所有文字使用繁體中文
